# 西洋星座分析

## When to Use This Skill

当用户要求以下任务时使用：
- 查询星座 / 什么星座
- 星座性格分析
- 星座配对 / 星座合盘
- 星座相位分析
- 任何涉及十二星座的问题

## Instructions

### 第一步：确定星座

根据公历出生月日确定星座：

| 星座 | 日期范围 | 元素 | 模式 | 守护星 |
|------|----------|------|------|--------|
| 白羊座 | 3/21-4/19 | 火 | 开创 | 火星 |
| 金牛座 | 4/20-5/20 | 土 | 固定 | 金星 |
| 双子座 | 5/21-6/21 | 风 | 变动 | 水星 |
| 巨蟹座 | 6/22-7/22 | 水 | 开创 | 月亮 |
| 狮子座 | 7/23-8/22 | 火 | 固定 | 太阳 |
| 处女座 | 8/23-9/22 | 土 | 变动 | 水星 |
| 天秤座 | 9/23-10/23 | 风 | 开创 | 金星 |
| 天蝎座 | 10/24-11/22 | 水 | 固定 | 冥王星 |
| 射手座 | 11/23-12/21 | 火 | 变动 | 木星 |
| 摩羯座 | 12/22-1/19 | 土 | 开创 | 土星 |
| 水瓶座 | 1/20-2/18 | 风 | 固定 | 天王星 |
| 双鱼座 | 2/19-3/20 | 水 | 变动 | 海王星 |

### 第二步：分析特质

为每个星座提供以下信息：
1. **元素**：火/土/风/水
2. **模式**：开创/固定/变动
3. **守护星**及其影响
4. **核心特质**（正面和需注意的方面）
5. **适合职业方向**
6. **情感模式**

详见 `references/zodiac-details.md`。

### 第三步：配对/合盘分析

计算两个星座之间的角度差（每个星座30°）：

| 角度 | 相位名称 | 评价 | 匹配度 |
|------|----------|------|--------|
| 0° | 合相 | 性格相似，天然理解 | ★★★★★ |
| 30° | 半六合 | 温和协调 | ★★★★☆ |
| 60° | 六合 | 和谐互助 | ★★★★★ |
| 90° | 四分相 | 有摩擦和张力 | ★★★☆☆ |
| 120° | 三合 | 非常和谐 | ★★★★★ |
| 150° | 梅花相 | 需要调整适应 | ★★☆☆☆ |
| 180° | 对冲 | 互补但需磨合 | ★★★☆☆ |

### 元素相性

| 组合 | 关系 |
|------|------|
| 火+火 | 热情高涨，但易冲突 |
| 火+风 | 风助火势，相互激励 |
| 火+土 | 火生土，滋养型关系 |
| 火+水 | 水克火，需要磨合 |
| 土+土 | 稳定踏实，可能沉闷 |
| 土+水 | 包容滋润型 |
| 土+风 | 理念差异较大 |
| 风+风 | 沟通顺畅，活跃 |
| 风+水 | 理智与感性结合 |
| 水+水 | 感性深邃 |

### 输出格式

```
【星座分析】[星座名]
  元素：[火/土/风/水]
  守护星：[行星名]
  模式：[开创/固定/变动]
  核心特质：[关键词]
  
【配对分析】
  [星座A] ↔ [星座B]
  相位：[相位名]（[角度]°）
  匹配度：[星级]
  分析：[具体解读]
```

## Important Notes

- 星座只是太阳星座，完整星盘还需要月亮、上升等
- 边界日期的人可能同时具有两个星座特质（"边界星座"）
- 星座分析仅供娱乐参考
