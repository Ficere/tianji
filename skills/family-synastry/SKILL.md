# 家庭/团队合盘综合评估

## When to Use This Skill

当用户要求以下任务时使用：
- 家庭合盘 / 家人命理配对
- 夫妻合盘 / 婚姻配对
- 团队成员命理兼容性评估
- 多人八字对比分析
- 任何涉及两人或多人命理关系的评估

## Prerequisites

在使用此技能之前，需要先完成每个人的个人测算。建议加载以下技能：
- `bazi-analysis`：八字五行测算
- `bone-weight-fortune`：袁天罡称骨
- `ziwei-chart`：紫微排盘
- `zodiac-analysis`：星座分析

## Instructions

### 第一步：收集每个人的测算结果

确保已获得每个人的：
1. 四柱八字和五行分布
2. 日主（天干）
3. 年支（生肖）
4. 星座
5. 称骨重量
6. 紫微命宫概要

### 第二步：五行互补分析 (权重25%)

1. 列出每人的五行分布表
2. 计算家庭/团队的五行合计
3. 评估平衡度（最大值-最小值）
4. 检查是否五行俱全
5. 分析互补关系（A缺什么，B是否能补）

**评分标准**：
- 差异 ≤ 3：满分
- 差异 4-5：良好
- 差异 6-8：一般
- 差异 > 8：偏差较大
- 五行俱全额外加分

### 第三步：生肖关系分析 (权重20%)

检查成员之间的生肖关系：

**吉利关系**（加分）：
- **六合**：子丑、寅亥、卯戌、辰酉、巳申、午未（极佳，+7分/对）
- **三合**：申子辰、寅午戌、巳酉丑、亥卯未（好，+5分/对）

**不利关系**（减分）：
- **六冲**：子午、丑未、寅申、卯酉、辰戌、巳亥（-5分/对）
- **相害**：子未、丑午、寅巳、卯辰、申亥、酉戌（-3分/对）
- **相刑**：子卯、丑戌未、寅巳申（-2分/对）
- **相破**：子酉、丑辰、寅亥、卯午、巳申、未戌（-1分/对）

**三合局检测**：检查三个成员是否能组成完整三合局。

### 第四步：星座合盘分析 (权重15%)

计算每对成员之间的星座相位：

| 角度 | 类型 | 得分 |
|------|------|------|
| 0° | 合相 | +5分 |
| 60° | 六合 | +5分 |
| 120° | 三合 | +5分 |
| 30° | 半六合 | +4分 |
| 180° | 对冲 | +3分 |
| 90° | 四分 | +2分 |
| 150° | 梅花 | +1分 |

分析家庭元素组合（火/土/风/水的分布）。

### 第五步：日主生克分析 (权重20%)

分析每对成员的日主五行关系：

1. **天干合**（极佳，+7分/对）：甲己化土、乙庚化金、丙辛化水、丁壬化木、戊癸化火
2. **相生**（好，+5分/对）：木生火、火生土、土生金、金生水、水生木
3. **比和**（平，+5分/对）：同五行
4. **相克**（需注意，+1分/对）：木克土、土克水、水克火、火克金、金克木

### 第六步：称骨对比分析 (权重20%)

1. 列出每人的骨重和等级
2. 计算家庭平均骨重
3. 评价家庭整体命格

| 平均骨重 | 评价 | 得分 |
|----------|------|------|
| ≥4.5两 | 中上，衣食无忧 | 16分 |
| 4.0-4.4两 | 较好 | 14分 |
| 3.5-3.9两 | 中等偏上 | 12分 |
| 3.0-3.4两 | 中等 | 9分 |
| < 3.0两 | 偏轻 | 5分 |

### 第七步：综合评分

将各维度得分汇总，总分100分。

| 评分范围 | 评级 |
|----------|------|
| 85-100 | ★★★★★ 极佳组合 |
| 70-84 | ★★★★☆ 良好组合 |
| 55-69 | ★★★☆☆ 中等组合 |
| 40-54 | ★★☆☆☆ 有待改善 |
| < 40 | ★☆☆☆☆ 需要多加注意 |

### 第八步：注意事项与建议

1. 列出需要注意的不利因素
2. 针对每个不利因素给出改善建议
3. 列出家庭优势和积极因素
4. 给出整体建议

### 输出格式

参考 `references/output-format.md`。

## Important Notes

- 合盘分析是综合性评估，单一不利因素不代表整体不好
- 所有命理分析仅供参考娱乐
- 人际关系的好坏主要取决于沟通、理解和努力
- 评分体系为简化模型，不同流派可能有不同看法
