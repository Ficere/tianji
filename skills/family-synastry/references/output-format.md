# 合盘输出格式模板

## 标准输出结构

```
╔══════════════════════════╗
║  家庭合盘综合分析报告    ║
╚══════════════════════════╝

一、家庭五行互补分析
  [五行分布表格]
  [五行图形化分布]
  [互补关系分析]

二、生肖关系分析
  [每对成员的生肖关系]
  [三合/六合局检测]

三、星座合盘分析
  [每对成员的星座相位]
  [元素组合分析]

四、日主生克关系
  [每对成员的日主关系]
  [天干合化分析]

五、称骨重量对比
  [骨重对比表]
  [家庭平均骨重]

六、家庭综合评分
  [评分明细]
  [总分和评级]

七、注意事项与建议
  ⚠️ 需要注意的方面
  ✅ 家庭优势
  💡 改善建议
```

## 关系符号说明

- ✅ 吉利/有利关系
- ⚠️ 需注意/不利关系
- ➖ 无特殊关系
- 🎯 天干合（极佳缘分）
- ★ 命宫标记
- ☆ 身宫标记
