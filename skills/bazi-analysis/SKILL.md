# 八字五行测算

## When to Use This Skill

当用户要求以下任务时使用：
- 排八字 / 排四柱
- 分析五行 / 五行缺什么
- 查看天干地支
- 十神分析
- 纳音五行查询
- 日主旺衰分析
- 任何涉及"生辰八字"、"命理"的测算

## Instructions

### 第一步：确认信息

1. 需要用户提供：公历出生年月日 + 出生时间（精确到时辰即可）
2. 确认是否为真太阳时（若用户未说明，默认为北京时间，提示用户真太阳时可能有偏差）
3. 确认性别（影响大运方向）

### 第二步：农历转换

将公历日期转换为农历日期，需确认：
- 农历年份（以立春为界，不是春节）
- 农历月份（注意闰月处理）
- 农历日期

**关键节气分界**（月柱以节气为界）：
| 月支 | 节气起点 | 大约公历 |
|------|----------|----------|
| 寅（正月）| 立春 | 2月4日前后 |
| 卯（二月）| 惊蛰 | 3月6日前后 |
| 辰（三月）| 清明 | 4月5日前后 |
| 巳（四月）| 立夏 | 5月6日前后 |
| 午（五月）| 芒种 | 6月6日前后 |
| 未（六月）| 小暑 | 7月7日前后 |
| 申（七月）| 立秋 | 8月8日前后 |
| 酉（八月）| 白露 | 9月8日前后 |
| 戌（九月）| 寒露 | 10月8日前后 |
| 亥（十月）| 立冬 | 11月7日前后 |
| 子（冬月）| 大雪 | 12月7日前后 |
| 丑（腊月）| 小寒 | 1月6日前后 |

### 第三步：排四柱

#### 年柱
- 以立春为年份分界（不是春节）
- 天干：(年份 - 4) % 10 对应甲乙丙丁戊己庚辛壬癸
- 地支：(年份 - 4) % 12 对应子丑寅卯辰巳午未申酉戌亥

#### 月柱
- 地支：由节气决定（见上表）
- 天干：五虎遁口诀
  - 甲己之年丙作首（寅月起丙）
  - 乙庚之岁戊为头（寅月起戊）
  - 丙辛之岁寻庚上（寅月起庚）
  - 丁壬壬寅顺水流（寅月起壬）
  - 戊癸何方发甲寅（寅月起甲）

#### 日柱
- 必须查万年历或使用在线排盘工具验证
- 建议访问可靠的排盘网站进行交叉验证

#### 时柱
- 地支：由出生时辰决定
  | 时辰 | 时间段 |
  |------|--------|
  | 子时 | 23:00-01:00 |
  | 丑时 | 01:00-03:00 |
  | 寅时 | 03:00-05:00 |
  | 卯时 | 05:00-07:00 |
  | 辰时 | 07:00-09:00 |
  | 巳时 | 09:00-11:00 |
  | 午时 | 11:00-13:00 |
  | 未时 | 13:00-15:00 |
  | 申时 | 15:00-17:00 |
  | 酉时 | 17:00-19:00 |
  | 戌时 | 19:00-21:00 |
  | 亥时 | 21:00-23:00 |
- 天干：五鼠遁口诀
  - 甲己还加甲（子时起甲）
  - 乙庚丙作初（子时起丙）
  - 丙辛从戊起（子时起戊）
  - 丁壬庚子居（子时起庚）
  - 戊癸壬子头（子时起壬）

### 第四步：分析

1. **五行统计**：统计八个字（4天干+4地支）的五行分布
2. **五行旺衰**：结合出生季节判断日主旺衰
3. **十神分析**：以日干为中心，分析其他七个字与日干的关系
4. **纳音五行**：查60甲子纳音表
5. **地支藏干**：列出每个地支所藏的天干
6. **喜用神**：根据日主旺衰和五行平衡推断喜用神（简化版）

### 第五步：输出格式

参考 `references/output-template.md` 中的标准输出格式。

## Important Notes

- 日柱计算复杂，务必通过在线工具或万年历验证
- 以立春为年份分界，而非春节
- 月柱以节气为分界，而非农历初一
- 真太阳时与北京时间可能有差异（尤其是西部地区）
- 测算结果仅供参考娱乐，命理学并非精确科学
