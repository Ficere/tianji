# 八字五行测算输出模板

## 标准输出结构

```
┌──────────────────────────────────────────┐
│              四 柱 八 字                  │
├──────────┬──────────┬──────────┬──────────┤
│   年柱   │   月柱   │   日柱   │   时柱   │
├──────────┼──────────┼──────────┼──────────┤
│   [年GZ] │  [月GZ]  │  [日GZ]  │  [时GZ]  │
├──────────┼──────────┼──────────┼──────────┤
│  [五行]  │  [五行]  │  [五行]  │  [五行]  │
├──────────┼──────────┼──────────┼──────────┤
│  [纳音]  │  [纳音]  │  [纳音]  │  [纳音]  │
└──────────┴──────────┴──────────┴──────────┘
```

### 包含内容

1. **基本信息**
   - 公历/农历日期
   - 生肖
   - 纳音命

2. **四柱八字**（表格形式）
   - 天干地支
   - 五行属性
   - 纳音五行

3. **十神分析**
   - 各柱天干对应十神
   - 十神含义简述

4. **地支藏干**
   - 每个地支所藏天干及其五行

5. **五行统计**
   - 金木水火土各几个
   - 五行最旺/最弱
   - 五行缺否
   - 图形化分布

6. **日主分析**
   - 日主五行和阴阳
   - 出生季节
   - 旺衰初判

7. **喜用神建议**（简化版）
