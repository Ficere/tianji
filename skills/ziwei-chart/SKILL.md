# 紫微斗数排盘

## When to Use This Skill

当用户要求以下任务时使用：
- 紫微斗数排盘
- 紫微命盘分析
- 查看命宫、身宫
- 十二宫位分析
- 紫微主星分布
- 任何涉及"紫微"、"斗数"的测算

## Instructions

### 前置条件

需要用户提供：
1. 农历出生年月日（或公历转换）
2. 出生时辰
3. 性别（决定大运方向）

### 排盘步骤

#### 1. 安命宫

口诀：**斗柄建寅起正月，数至生月顺流行，子时起数生时止，逆回安命顺安身**

具体操作：
1. 从寅宫起正月，顺时针数到出生农历月份
2. 从该宫起子时，逆时针数到出生时辰 → 此宫为**命宫**
3. 从该宫起子时，顺时针数到出生时辰 → 此宫为**身宫**

公式：
- 命宫地支索引 = (寅的索引 + 月份 - 1 - 时辰索引) % 12
- 身宫地支索引 = (寅的索引 + 月份 - 1 + 时辰索引) % 12

#### 2. 定十二宫

从命宫开始，逆时针依次排列：
命宫 → 兄弟宫 → 夫妻宫 → 子女宫 → 财帛宫 → 疾厄宫 → 迁移宫 → 交友宫 → 官禄宫 → 田宅宫 → 福德宫 → 父母宫

#### 3. 安命宫天干（五虎遁）

根据出生年天干，确定寅宫的天干，然后顺推至命宫：
| 年干 | 寅宫起 |
|------|--------|
| 甲/己 | 丙寅 |
| 乙/庚 | 戊寅 |
| 丙/辛 | 庚寅 |
| 丁/壬 | 壬寅 |
| 戊/癸 | 甲寅 |

#### 4. 定五行局

根据命宫的天干地支，查纳音确定五行局：
- 水二局、木三局、金四局、土五局、火六局

详见 `references/ziwei-tables.md`。

#### 5. 定命主和身主

**命主**（由命宫地支决定）：
| 命宫地支 | 命主星 |
|----------|--------|
| 子 | 贪狼 | 丑 | 巨门 | 寅 | 禄存 | 卯 | 文曲 |
| 辰 | 廉贞 | 巳 | 武曲 | 午 | 破军 | 未 | 武曲 |
| 申 | 廉贞 | 酉 | 文曲 | 戌 | 禄存 | 亥 | 巨门 |

**身主**（由出生年地支/生肖决定）：
| 年支 | 身主星 |
|------|--------|
| 子 | 铃星 | 丑 | 天相 | 寅 | 天梁 | 卯 | 天同 |
| 辰 | 文昌 | 巳 | 天机 | 午 | 火星 | 未 | 天相 |
| 申 | 天梁 | 酉 | 天同 | 戌 | 文昌 | 亥 | 天机 |

#### 6. 定大运方向

- 阳年男命 / 阴年女命 → 顺行
- 阴年男命 / 阳年女命 → 逆行

阳年天干：甲丙戊庚壬
阴年天干：乙丁己辛癸

#### 7. 安紫微星及十四主星

以五行局和农历出生日数来推算紫微星的位置，再由紫微星位推算其余十三颗主星。
此步骤较为复杂，详见 `references/ziwei-tables.md`。

### 输出格式

```
【紫微斗数排盘概要】
  命宫：[天干][地支]宫
  身宫：[地支]宫（落在[宫名]）
  五行局：[X行X局]
  命主：[星名]
  身主：[星名]
  [阴阳]，[性别]命，大运[方向]

  十二宫位分布：
    命宫：[地支]宫 ★
    兄弟宫：[地支]宫
    ...
```

## Important Notes

- 紫微斗数排盘非常复杂，完整排盘需要安放约100颗星
- 本技能提供概要排盘（命身宫、十二宫、五行局、命身主等基本信息）
- 建议对于详细排盘，引导用户使用专业排盘软件进行验证
- 农历闰月的处理有多种流派，常见做法是15日前取当月，16日后取下月
