# 袁天罡称骨算命

## When to Use This Skill

当用户要求以下任务时使用：
- 袁天罡称骨 / 称骨算命
- 查骨重 / 八字有多重
- 称骨歌诀解读
- 任何涉及"称骨"、"骨重"的测算

## Instructions

### 第一步：确认农历信息

需要将用户的出生信息转换为农历：
1. 农历年份的干支（如癸酉年、甲戌年等）
2. 农历月份（正月到腊月）
3. 农历日期（初一到三十）
4. 出生时辰（子时到亥时）

**注意**：年份以立春为界，月份以节气为界。

### 第二步：查表计算骨重

分别查出年、月、日、时四项骨重，然后相加。

#### 年份骨重

查 `references/weight-tables.md` 中的年份对照表。以六十甲子为周期，相差60年的年份骨重相同。

#### 月份骨重

| 月份 | 骨重 |
|------|------|
| 正月 | 6钱 |
| 二月 | 7钱 |
| 三月 | 1两8钱 |
| 四月 | 9钱 |
| 五月 | 5钱 |
| 六月 | 1两6钱 |
| 七月 | 9钱 |
| 八月 | 1两5钱 |
| 九月 | 1两8钱 |
| 十月 | 8钱 |
| 十一月 | 9钱 |
| 十二月 | 5钱 |

#### 日期骨重

查 `references/weight-tables.md` 中的日期对照表。

#### 时辰骨重

| 时辰 | 时间段 | 骨重 |
|------|--------|------|
| 子时 | 23:00-01:00 | 1两6钱 |
| 丑时 | 01:00-03:00 | 6钱 |
| 寅时 | 03:00-05:00 | 7钱 |
| 卯时 | 05:00-07:00 | 1两 |
| 辰时 | 07:00-09:00 | 9钱 |
| 巳时 | 09:00-11:00 | 1两6钱 |
| 午时 | 11:00-13:00 | 1两 |
| 未时 | 13:00-15:00 | 8钱 |
| 申时 | 15:00-17:00 | 8钱 |
| 酉时 | 17:00-19:00 | 9钱 |
| 戌时 | 19:00-21:00 | 6钱 |
| 亥时 | 21:00-23:00 | 6钱 |

### 第三步：查歌诀

将总骨重（1两=10钱）在 `references/weight-tables.md` 中查找对应的歌诀和命理解释。

### 第四步：输出

输出格式：
```
【袁天罡称骨算命】
  年骨重：X两X钱（XX年）
  月骨重：X两X钱（农历X月）
  日骨重：X两X钱（XX日）
  时骨重：X两X钱（X时）
  ─────────────────────────
  总骨重：X两X钱
  等级：[命薄/命轻/中等/中上/上等/上上/极佳]
  歌诀：[对应诗句]
```

### 骨重等级参考

| 骨重范围 | 等级 |
|----------|------|
| 2两1-2两4 | 命极薄 |
| 2两5-2两8 | 命轻 |
| 2两9-3两5 | 中等 |
| 3两6-3两9 | 中等偏上 |
| 4两-4两4 | 中上 |
| 4两5-5两4 | 上等 |
| 5两5-5两9 | 上上 |
| 6两以上 | 极佳 |

## Important Notes

- 1两 = 10钱
- 骨重最轻约2两1钱，最重约7两2钱
- 歌诀范围为2两1到7两2
- 此法为古代命理术，仅供参考娱乐
