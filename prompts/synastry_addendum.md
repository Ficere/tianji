# 合盘场景补充提示

本文件在合盘（scene=synastry）时附加到步骤十的输出指令之后。

## SynastryReading 字段补充说明

### 三维评分

- `core_resonance.score`（/40）：评估两人内核层契合度——日主生克关系 + 命宫主星相性 + 月亮星座角度 + MBTI底层功能互补
- `will_synergy.score`（/35）：评估意志协同度——太阳星座角度 + 大限主题重叠度 + 财官格局匹配
- `outer_friction.score`（/25）：评估外在摩擦点——上升星座相位关系 + 五行显性差异 + 当前MBTI状态差异；**分值越高摩擦越大**

三维评分之和不等于总合盘分（composite_scores.total），两套体系独立，不得混用。

### composite_scores 评分细则

| 维度 | 满分 | 评分逻辑 |
|------|------|---------|
| wuxing_balance（五行平衡） | 20 | 两人五行互补程度：俱缺同一行扣分，互补加分 |
| wuxing_complete（五行俱全） | 5 | 合并后五行是否俱全：是=5，缺1行=3，缺2行=1 |
| shengxiao（生肖） | 20 | 三合=20，六合=16，无刑无冲=12，相害=6，相冲=2，相刑=0 |
| xingzuo（星座相位） | 15 | 0/60/120°=13-15；30°=10-12；180°=7-9；90°=4-6；150°=1-3 |
| riZhu（日主关系） | 20 | 同气/相生=16-20；相克但有化=8-15；直冲无化=0-7 |
| chenggu（称骨对比） | 15 | 均值骨重差异≤0.5两=13-15；差0.5-1两=9-12；差1-1.5两=5-8；差>1.5两=1-4 |
| xingming（姓名相合） | 5 | 三才配置相生=4-5；无关=2-3；相克=0-1；无姓名信息=不计入，total相应减5 |

### scenario_advice 场景选择

只输出与实际关系场景相关的建议，最少1个，最多覆盖3个最相关场景。不要每个场景都写（除非用户明确要求全部）。

enum 选项：`"情侣/伴侣"` `"亲子"` `"职场搭档"` `"团队协作"` `"管理者与下属"` `"朋友"`

### compatibility_summary_line

一句话金句，10-20字，捕捉这对组合的本质。风格举例：
- 「内核共鸣，外在需磨合——彼此是镜子」
- 「各有宿命，相遇是契机而非归宿」
- 「互补格局，但需学会说不同的语言」

避免使用：「天作之合」「命中注定」「绝配」等过于绝对的表达。
